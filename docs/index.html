Macro Microcontroller 2

Macro Microcontroller 2, a macropad + microcontroller development board that controls GPIO pin output using keys with a custom BASIC interpreter firmware.

Hardware Specifications

Dimensions: 96.52 × 35.56 mm
Microcontroller: RP2040
Flash Memory: 4MiB
Power: 3.3V
View schematics

Firmware

Macro Microcontroller BASIC is a custom BASIC interpreter for Macro Microcontroller 2. You can install it by holding the boot button and plugging it in to your computer. Download the following file and move it to the storage drive that appears.

Download macro-microcontroller-basic-0.0.1.uf2 via releases. View source code.
